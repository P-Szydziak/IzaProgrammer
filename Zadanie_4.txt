Opis:
Programista bez gita jak bez ręki.
Pobierz repozytorium do siebie lokalnie za pomocą komendy git clone.
Stwórz branch i wrzuć do niego projekt. Dodaj Git ignore i potem reszte projekty.

Wymagania:
- Pobierz projekt za pomocą git clone
- Stwórz własnego brancha za pomocą git brancha
- Przejdź na swojego nowego brancha (doczytaj jak)
- Wrzuć projekt do swojego brancha 
- Dodaj git ignore, aby nie wrzucić zbędnych plików (znajdź git ignore dla Visual Studio z C# na internecie)
- Dodaj najpierw commit z git ignore, a potem z resztą projektu

Podpowiedzi:
- użyj link z HTTPS, bo SSH będzie na ten moment za ciężkie
- kolejność commitów jest ważna!
